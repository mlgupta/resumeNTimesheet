Drawers/Folders:
Please note that "Drawer" and "Folder" are synonyms for the entire manual .
Drawers management is only available in Flat Navigation(see User Preferences Help for navigation type).
A non-administrator can only manage drawers corresponding to "My Docs".
An administrator can manage drawers across "Resumes", "Timesheets" and "My Docs".
After Logging in, select 'Drawers' icon [ ] to view your folder listing. testImg
 
Drawers Management
 
  • A non admin will only be able to view and work on Personal(read My Docs) drawers.
  • An admin can view and work on Personal(read My Docs), Resume and Timesheet drawers.
  • Click on the desired tab to view drawer listing.
