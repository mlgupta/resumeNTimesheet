How to edit an existing Timesheet ?
  • First go to the folder/drawer where you would like to edit an existing timesheet.
  • Select the desired resume and click on the 'Edit Timesheet' icon [ ].
  • You can also directly click on the desired timesheet for the edit page to appear.
  • Please note that whenever a versioned timesheet is edited successfully, a new version with implicit check out and check in is created.
  • Aborting an edit operation for a versioned timesheet doesnot create a new version and the latest version is restored.
  • The edit page appears with prepopulated entries of timesheet stored.
  • In the case of versioned timesheets, the edit page appears with prepopulated entries of the most recent version.
