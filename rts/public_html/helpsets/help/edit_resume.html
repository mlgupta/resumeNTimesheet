How to edit an existing Resume ?
  • First go to the folder/drawer where you would like to edit an existing resume.
  • Select the desired resume and click on the 'Edit Resume' icon [ ].
  • You can also directly click on the desired resume for the edit page to appear.
  • Please note that whenever a versioned resume is edited successfully, a new version with implicit check out and check in is created.
  • Aborting an edit operation for a versioned resume doesnot create a new version and the latest version is restored.
  • The edit page appears with prepopulated entries of resume stored.
  • In the case of versioned resumes, the edit page appears with prepopulated entries of the most recent version.
