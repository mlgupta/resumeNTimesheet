What is User Preference?
  • User Preference is the profile of the logged-in user where information related to his profile is displayed.
  • The preferences of the logged-in user can be set like
    • 'Login Password'.
    • 'Encrypt/Decrypt Password'.
    • 'Permitted tree access level'.
    • 'Items to be displayed per page'.
    • 'Themes'.
    • 'Open Document In New Window '.

Login Password - - Facilitates the logged-in user to change the password.
  • Click on the 'Change Login Password' button.
  • Change password page will be displayed.
  • Enter the 'Password' and 'Confirm Password'.
  • Press 'OK' to change the password.
  • Press 'Cancel' to abort.
