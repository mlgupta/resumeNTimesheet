How to Cut Folder,Document,Resume or Timesheet?
  • First select the folder,document,resume or timesheet you would like to cut.
  • You can select multiple folder,document,resume or timesheet.
  • Click on the 'Cut' icon .
  • Successful cut will give the message as 'Cut Action performed successfully' in the Task bar.
 
How to Copy Folder,Document,Resume or Timesheet?
  • First select the folder,document,resume or timesheet you would like to copy.
  • You can select multiple folder,document,resume or timesheet.
  • Click on the 'Copy' icon .
  • Successful copy will give the message as ' Copy action performed successfully' in the Task bar.
How to Paste Folder,Document,Resume or Timesheet?
  • First goto the folder where you would like to paste.
  • Click on the 'Paste' icon .
  • Successful Paste will give the message as 'The selected items were pasted successfully' in the Task bar.
