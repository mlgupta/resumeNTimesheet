How to Copy Folder,Document,Resume or Timesheet to other Folder?
  • First select the folder,document,resume or timesheet you would like to copy.
  • You can select multiple folder,document,resume or timesheet.
  • Click on the 'Copy To' icon [ ].
  • Select the folder where you would like to Copy the selected items.
  • Press 'Select' to copy the selected items to the selected Folder.
  • Press 'Cancel' to abort copying.
  • Press 'New Folder' to create New Folder.

How to Move Folder,Document,Resume or Timesheet To Folder?
  • First select the folder,document,resume or timesheet you would like to move.
  • You can select multiple folder,document,resume or timesheet.
  • Click on the 'Move To' icon [ ].
  • Select the folder where you would like to Move the selected items.
  • Press 'Select' to move the selected items to the selected Folder.
  • Successful moving to the selected folder will give message as 'The selected items were moved successfully'.
  • Press 'Cancel' to abort moving.
  • Press 'New Folder' to create New Folder.
