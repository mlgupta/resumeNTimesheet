How To Edit a User?
  • You can Edit one user at a time.
  • Click on 'Edit User' Icon [ ] from the User List, Edit User Page appears with previously saved data.

General User Information:
Name Min Length Max Length Description Required /Optional
User Name 5 20 Unique Account Name for the user. (Entries for the Home Folder and Mail folder for this user are derived from the name.) Non-Editable
Description - - User's description. Optional
Password 5 20 Password for the user. Required
Confirm Password 5 20 Password for the user. Required
Email Address  Enter email address constructed from the user name, plus an "@" symbol, plus the fully qualified company name. (suzy.jones@yourcompany.com) Optional
'Save' button  Press 'Save' to save the changes done on the User, the User will be saved and User list will appear with the message in the status Bar as "User 'Steffia' successfully saved".  
'Cancel' button  Press 'Cancel' to abort changes done on User.  

Note
  • A user's home folder is the default directory where a user starts when entering into the My Docs module.
