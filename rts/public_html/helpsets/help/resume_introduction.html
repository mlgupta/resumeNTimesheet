Resumes:
Resumes are managed through folders in a structured way.
After Logging in, select Resume icon [ ] to view the folders / resumes listing for your company.
 
Resume/Folder page consists of following parts.
  • Tool-bar consists of icons which assists you to do various operations on Folder/Resume List.
  • Address-bar assists you to navigate through the folders by directly entering the path of the folder
    separated by '/ '.
  • Navigation Panel consists of Tree view which lists the folders.
    You can navigate through the folders by selecting the folders icon.
  • Folder/Resume List Panel consists of listing of folder and resumes.
  • Search Panel consists of various search criteria's. You can search for the resumes as well as folders.
  • Status-bar consists of two parts
    • Message-bar: displays the message of the operations performed on the Folder/Resume List.
    • Navigation-bar: helps in navigating through the result when the total items are more than
      the displayed items

How to use Address bar?
  • Enter the path where you would like to navigate.
  • Press 'Go' icon to go to display the record of the entered path.
  • Folder/Resume will be displayed corresponding to the path entered.

How to use Navigation Panel?
  • '/ ' indicates root folder.
  • '-' indicates expanded state of the folder. Clicking on it will hide the sub folders.
  • '+' indicates collapsed state of the folder. Clicking on it will display the sub folders.
  • Clicking on the folder will display the contents of the folder in the Folder/Resume List on the right-hand side.
  • Open Folder indicates that the contents of the folder are displayed in the Folder/Resume List and vice-versa.
 
How to use Folder/Resume List?
*Above displayed icons are:
  • Folder [ ]
  • Resume [ ]
  • Versioned Resume [ (v) ]

1) To see the content of the folder click on the text of the corresponding icon.

2) To edit a resume click on the text of the corresponding icon.

3) To view the folder above the displayed folder click up icon.

4) You can navigate through the Folder/Resume List using the Previous, Next, Up icons, to view the corresponding pages.
